Use this service to:

  • register trainee data for the census
  • request a teacher reference number (TRN)
  • recommend trainees for QTS or EYTS
  • check and update your trainee data

If you are a Higher Education Institute (HEI), you must submit your data using the Register trainee teacher service. You do not submit it via the Higher Education Statistics Agency (HESA).

You can only register trainee teachers for the current academic year. If you need to register a trainee teacher within a different academic year, contact itt.datamanagement@education.gov.uk.

Assessment Only (AO) route guidance

To find out about AO and census timescales, you can view AO guidance.

Register trainee teachers data requirements

We have reduced the amount of data we ask for. To help you collect and prepare your data, you can view route specific data requirements.
